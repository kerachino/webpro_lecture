How to use this website
サイトの使い方
当サービスを利用した場合、"プライバシーポリシー"、"ウェブサイトについて"に同意したものとみなします。
当サイトはいつでもどこでもだれでも、英単語が覚えられるようにと開発された英単語帳アプリです。

紙の単語帳のように、覚えたい英単語とその意味を登録してランダムにテストすることができます。
スマートフォン、PCのどちらにも対応していますので、英単語の勉強に活用してください。
英単語テストについて
英単語のテストはエビングハウスの忘却曲線に基づいて出題しています。例えば当日と2日前のもの、7日前のものからランダムに出題するようになります。すべてからの出題も可能。
音声が再生されたり、自動テスト実行ボタンもあるのでご活用ください。
過去全ての英単語を見たい場合には"ALL"ボタンを押してください。
英単語の登録・編集について
英単語、意味、メモ、登録した日付の4つを保存することができ、日付を除き、テストを行う際に表示されます。
また、登録した英単語は編集、削除、そしてキーワード検索を行った後に単語の編集、削除を行うことができます。
なお、日付は英単語テストを出題する際に使用します。詳しくは英単語テストについてを伽藍ください。
キーワード検索は重複する物がある場合、右側の矢印で次に進むことができます。!マークにカーソルを合わせると詳細が表示されます。
教育モード
現在調整中ですが、一部の機能をご利用いただけます。勝手に単語の削除をできないようにパスワードを設ける機能です。
今後は文字をひらがなにするなど、お子様でも簡単に操作できるような工夫をする予定です。
初めてご使用される際は、教育モードボタンを押した後に、任意のパスワードを設定してください。
なお、設定したパスワードに関しては外部からはアクセスできませんが、端末内では確認することができます。ご了承ください。
(お使いのPCやスマートフォンを第三者が操作できる場合は、見ようと思えばパスワードを見ることができます)
昨日忘れた、何日戻る、日付指定ボタンについて
英単語のテストはエビングハウスの忘却曲線に基づいて出題しています。
色覚障害の方へ
左上にある歯車マークを押すと、色補正のボタンがあります。多少文字が見えやすくなります。
また、シンプルモードもあるので、使いやすいほうでご活用ください。
Q&A

ローカルストレージの削除行ってください。なお、お使いのブラウザにより操作が異なるため、方法は割愛させていただきます。
どうしても分からない場合はお問い合わせください。
